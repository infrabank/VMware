# RCA 교차 검증 최종 평가 (CS01 AD Connectivity)

## 1. 총평

- 완성도: **높음 (90점 이상)**
- 상태: 운영 보고서 / 고객 보고 / 감사 대응 모두 사용 가능
- 한계: **근본 원인(프로세스 레벨) 미확정**

## 2. 핵심 개선 사항

### 2.1 인과관계 완성

```
포트 고갈 → RPC 실패 → Secure Channel 우회 → NTLM fallback → AD 차단
```

- Microsoft 내부 동작과 일치
- 기술적으로 정확

---

### 2.2 Event 5827 해석

- 5827 = 원인 ❌
- 5827 = 결과 (보안 정책 정상 동작) ✔

---

### 2.3 주기성 장애 반영

- 동일 시간대 반복 발생
- 주기적 작업 존재 가능성 반영됨

---

### 2.4 재발 예측 포함

- 재발 예상: 11~14일
- 운영 관점에서 매우 유의미

### 2.5 5 Whys 구조

- 구조적 설명 완성됨

## 3. 남은 문제 (핵심)

### 3.1 누수 vs 과부하 미구분

현재:
```
TIME_WAIT 누적
```

문제:
- 정상 환경에서도 발생 가능
- CS01만 문제 발생한 이유 설명 부족

필수 추가:
```
CS02와 비교 데이터 필요
```

---

### 3.2 VMware Tools 영향

현재:
- LOW 가능성

판단:
- 분석에서 제거 가능
- 근거 부족

---

### 3.3 근본 원인 정의 오류

현재:
```
TCP/IP 스택 + 포트 부족 = 근본 원인
```

정확한 구조:
```
근본 원인 = 비정상 포트 점유 (프로세스 미확정)
환경 요인 = 포트 제한 + OS 설정
```
