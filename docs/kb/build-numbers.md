## Quick Lookup

```bash
# ESXi build number check
esxcli system version get

# vCenter build number check (VCSA SSH)
vpxd -v

# PowerCLI
Get-VMHost | Select Name, Version, Build
$global:DefaultVIServer | Select Name, Version, Build
```
